\subsection*{Core Reminders}
\begin{itemize}[leftmargin=1.5em]
  \item \textbf{Bradley--Terry form:}
  \[
    P(y^+ \succ y^- \mid x)=\sigma(r(x,y^+)-r(x,y^-)).
  \]
  \item \textbf{KL regularization intuition:} keep policy close to a reference while improving target behavior.
  \item \textbf{Pairwise loss behavior:} margins matter more than absolute scores.
  \item \textbf{Support condition:} log-ratios are unstable if reference assigns near-zero mass.
\end{itemize}
