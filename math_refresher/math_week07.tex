\section{Week 7 Refresher: Low-Rank Adaptation and Quantization}

\subsection*{Setting the Scene}
This week is mostly two ideas: constrain update rank, and constrain numeric precision. Both are approximation tools with explicit tradeoffs.

\subsection*{Core Reminders}
\begin{itemize}[leftmargin=1.5em]
  \item \textbf{LoRA update form:}
  \[
    \Delta W = \frac{\alpha}{r}BA,\quad B\in\mathbb{R}^{d\times r},\ A\in\mathbb{R}^{r\times k}.
  \]
  \item \textbf{Trainable parameter fraction:}
  \[
    \rho=\frac{r(d+k)}{dk}.
  \]
  \item \textbf{Quantization view:} real value $\to$ codebook/scale representation.
  \item \textbf{Error tradeoff:} lower precision saves memory and bandwidth, increases approximation error.
\end{itemize}
